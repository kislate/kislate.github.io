　　从童年的小小贪吃蛇，俄罗斯方块，到4399，奥奇传说，再到后来迷上动漫，接触更多的游戏...我体验过欢欣和喜悦，感受过辛酸和苦楚。 那时候我也不清楚自己所痴迷些什么，或许是出于对通关的执着，或许是对那小小的收集爱好，有或许...仅仅是处于那幼稚的对父母的叛逆。可是随着长大，我渐渐地发现，游戏是是一片天地，动漫里有人生， 可我的生活不只有游戏。现如今回顾那些我曾喜欢的，所痴狂的，业已在记忆里淡化。但是它们曾经所给我带来的震撼，却时常能为我缓缓尘滞的心换一刹的悸动。因为它们曾装点了我梦 让我看到人的情感为何，我所追求为何，也让我知道了，我该怎样去生活。

这里是一些我所喜欢的游戏和动漫

空洞骑士（Hollow Knight）
　　游戏故事讲述一位小骑士所进行的冒险，他前往探查一座因瘟疫而荒废多时的地下昆虫王国圣巢（Hallownest）。
　　小骑士在旅途中探索未知地区、遇到各式各样的或敌或友的昆虫、 击败数十个Boss、学习技能与法术、收集物品和护符，在片段式的文字中解读各处的历史、 揭开王国的秘密，并在过程中了解自身的使命，最后直面瘟疫的源头
　　这款游戏是我在高考后暑假所接触的，是由澳大利亚独立团队樱桃游戏工作室 （Team Cherry）开发的一款 2D 类银河恶魔城动作冒险游戏。
　　虽然这款游戏剧情丰富，彩蛋多样，但实际上开发团队只是由三个人组成
　　之所以推荐它，是因为我现在还一直在玩时常在闲暇时间打开看一下。
　　这款游戏还有一个备受瞩目的续作《空洞骑士：丝之歌》（Hollow Knight: Silksong），只是至今都没出。






《艾尔登法环》（エルデンリング,lden Ring
　　这款游戏所获奖项可谓数不胜数：
　　British Academy Games Awards，Deutscher Computerspielpreis，D.I.C.E. Awards，Dragon Awards，Game Audio Network Guild Awards，The Game Awards，Game Developers Choice Awards， Gamescom，Golden Joystick Awards，Golden Trailer Awards，Japan Game Awards，MTV Millennial Awards，New York Game Awards........

　　这款游戏我在刚发售的时候就已经开始玩了，知道最近新出的《黄金树幽影》我也是第一是时间就通关了。确实属于一部空前绝后的佳作

《进击的巨人》
　　《进击的巨人》（日语：進撃の巨人）是日本漫画家谏山创创作的漫画作品。 漫画于2009年9月至2021年4月间在讲谈社《别册少年Magazine》上连载， 单行本全34卷。故事建立在人类与巨人之间的冲突，人类居住在由高墙包围的城市， 对抗墙外会吃人的巨人，并寻找著关于巨人的答案。
　　这是一部能够让我反复重温的神作，即使到现在我还能回想起当时的震撼
　　不过稍微有点遗憾的是这部动漫的结尾不是很令人满意，但也足以填满我的幻想
